# Timetrack

Timetrack is a single-file HTML/CSS/JavaScript web app for converting UNIX timestamps and customer-provided timezone references into local or selected timezones.

## Features

- UNIX timestamp conversion in seconds or milliseconds
- Timezone-to-timezone conversion
- Common timezone quick picks
- Human aliases such as India, Brazil, UTC, GMT, and Europe
- Theme picker with Dark, Daylight, Cyber Ember, Signal Grove, Sky Prism, and Yoho
- About and changelog modals
